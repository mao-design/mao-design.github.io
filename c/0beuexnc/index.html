进制的转换

  1. 二进制是由0和1组成的，由 0b 开头的
  2. 八进制是由 0 ~ 7 组成的，并且以 0 开头
  3. 十进制就是普通的数字，不需要加任何前缀
  4. 十六进制是 0 ~ 9 和 a ~ f 组成的，由 0x 开头（在十六进制中，a表示10，b表示11，以此类推，到 F 15为止）

细节：0xf0 其二进制

（十六进制）0xf == 15（十进制）
（十进制）15 == 1111（二进制）
（十六进制）0 == 0000（二进制）
所以 （十六进制）0xf0 == 11110000（二进制）

标识符的命名规则

  1. 组成字符：只能由字母（包括大写和小写）、数字和下划线（_）组成。

    例如，count、_value、num123 都是合法的标识符组成形式。

  2. 开头字符：必须以字母或者下划线开头，不能以数字开头

    比如 a1 是合法的，1a 则不合法。

  3. 不能与关键字重名：不能使用 C 语言的关键字（如 if、else、int、float、while 等）作为标识符，否则会引起编译错误。

  4. 区分大小写：C 语言对标识符大小写敏感，即 Max 和 max 被视为两个不同的标识符。

  5. 长度限制：C 标准没有规定标识符的最大长度，但各个编译器都有自己的限制。

    例如，有些编译器可能限制标识符长度为 31 个字符，实际使用中一般不用太担心，正常长度的标识符基本都能满足。

按位与 &

两个数都是1，结果才为1（1和1，结果才为1），否则为0（1和0，0和0，其结果都是0）

按位或 |

两位数中只要有1个1，其结果为1（1和0，1和1，其结果都为1），否则为0（0和0，结果为0）

按位异或 ^

两位数中是一个0加一个1组合的（1和0，结果为1），结果为1，否则为0（1和1，0和0，其结果都为0）

按位取反 ~

1变0，0变1

c = fgetc(file);
printf("%c", c);

// fgets  一次读一行，读不到返回 null
char zi[1024];
char *str;
str = fgets(zi, 1024, file);
printf("%s\n", str);

// fread  读取整个文件,当没有数据读取时，返回 0
char ch[1024];
int n;
// ch 将file中的值传到 ch 数组中； 1 表示每个字符的占用大小；
// 1024 表示读取一行字符的最大长度；file表示文件路径
n = fread(ch, 1, 1024, file);
printf("%d", n);
printf("%s", ch);

// 关闭文件：fclose
