进制

二进制是由0和1组成的，由 0b 开头的

八进制是由 0 ~ 7 组成的，并且以 0 开头

十进制就是普通的数字，不需要加任何前缀

十六进制是 0 ~ 9 和 a ~ f 组成的，由 0x 开头（在十六进制中，a表示10，b表示11，以此类推，到 F 15为止）
