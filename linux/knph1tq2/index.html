重置密码

140字小于1分钟

2025-05-28

修改密码

在该界面按 e

进去之后，找到以linux16开头的句子，在其末尾添加 init=/bin/sh

按 ctrl + x

输入 mount -o remount,rw /

按回车

在输入 passwd，修改密码，需要输入两次

输入 touch /.autorelabel

回车

最后，在输入 exec /sbin/init

回车，等待系统重启即可（时间可能会有点长）

进入系统，输入刚刚设置的密码即可
