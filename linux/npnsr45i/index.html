Ubuntu卸载软件

458字约2分钟

2024-12-02

参考Ubuntu 中怎么卸载 Linux 应用

方法一

在软件商店里面卸载软件，但是有些软件在软件商店中并没有列出来，这时候就需要使用命令行来卸载了

方法二

命令行卸载软件，安装应用时常用 apt-get install 或者 apt install 。 卸载应用时常用 apt-get remove 或者 apt remove

sudo apt remove 软件包名

技巧：如果你有些Linux命令记不得了，可以在终端中输入你记不得的那个命令的开头几个字母，然后再按键盘上的 Tab 键即可补全命令；软件也是一样的，记不得的软件名可以在终端中输入软件开头的几个字或则几个字母，然后在按 Tab 键即可补全

列出所有已安装的软件

apt list --installed

列出所有已安装的且包含my的软件

apt list --installed | grep -i my

补充

当在 Ubuntu 中卸载应用时，应用程序会被卸载，但是会留下细小的、修改过的用户配置文件。这些文件是故意被留下的，因为当你再次安装同样的应用时，会使用这些遗留的配置文件。

如果你想完全卸载应用，你可以使用 apt purge 命令代替 apt remove 命令，或者在 apt remove 命令后再使用它。

sudo apt purge 软件包名

注意 apt purge 命令不会删除保存在用户目录下的数据或者配置文件。

清理残留文件，比如不再用到的依赖或 Linux 内核头文件。

在终端输入如下命令：

sudo apt autoremove

这条命令很安全，而且会释放几百 MB 的磁盘空间。
