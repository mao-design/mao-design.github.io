Ubuntu安装搜狗输入法

发表于2022-04-09|更新于2024-03-05|Linux

安装搜狗输入法

  1. 下载搜狗输入法 Linux版的安装包（下载什么架构的包，按自己的处理器来决定，一般都是x86_64）

  2. 先卸载ibus输入法框架，命令是 sudo apt purge ibus

  3. 重启

  4. 安装fcitx输入法框架，命令是 sudo apt install fcitx ,然后在设置里面把fcitx设置为首选项，点开设置 --> 区域和语言 --> 管理已安装的语言，此时会弹出一个小窗口，显示 语言支持没有安装完整，点击详细信息，发现只有两个或者多个ibus软件，不用管，点击稍后提醒即可

    将键盘输入发系统设置为fcitx

    在启动fcitx即可

  5. 然后进入下载搜狗输入法软件包的文件夹里面，在该目录里面打开终端输入 sudo dpkg -i 搜狗输入法的软件包名（全称）

  6. 在安装输入法依赖

    然后重启即可，如果重启之后发现fcitx没有启动，那么就设置fcitx开机自启动，命令为 sudo cp /usr/share/applications/fcitx.desktop /etc/xdg/autostart/ ，就可以了，不出意外的话，搜狗输入法因该就可以使用了
