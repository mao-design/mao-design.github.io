美化Butterfly

发表于2024-02-06|Web

主题样式

在 css 中创建 _custom 文件夹，该文件夹是用来存放自己自定的css文件的

在_custom 文件夹中创建一个css文件，名字随意

修改css文件夹下的 index.styl，添加 @import '_custom/*.css' 到 // project 中的最后一行

字体

在 css 中创建 fonts 文件夹，该文件夹是用来存放自己自定的字体文件的

在_custom 文件夹中创建一个css文件，名字随意，然后引入 fonts 文件夹中的字体

修改css文件夹下的 index.styl，添加 @import '_custom/*.css'
