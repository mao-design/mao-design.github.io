Ubuntu星火软件推荐

发表于2022-04-15|更新于2024-03-04|Linux

安装星火软件商店

星火软件商店官网：星火应用商店

第一步，安装依赖，如果你是Deepin、UOS或者Ubuntu22.04系统，则不需要安装依赖文件

点击星火软件商店下载按钮，则会自动跳转到 gitee 上

注意：如果你是Deepin、UOS或者Ubuntu22.04系统，则不需要安装依赖文件，直接安装即可

官网下载的依赖包里面有两个文件夹，一个是Debian10和Ubuntu20.04的依赖包，一个是Debian11的依赖包，这两个依赖包文件夹里面都有一个 一键安装.sh 的脚本，在依赖包文件目录里面打开终端，输入 sudo ./一键安装.sh 即可运行 sh 脚本，这时依赖则会自动安装。

依赖安装完成之后再安装星火软件商店，先进入到星火软件商店软件包的下载目录里面，打开终端，输入 sudo apt install ./星火软件商店的全名 ，比如星火软件商店的软件包全名叫 spark-store_3.3.0.3_amd64.deb ，那么终端中输入的命令则是 sudo apt install ./spark-store_3.3.0.3_amd64.deb ,然后等待安装即可，安装完成之后就可以在星火软件商店里面安装软件了，也再此感谢那些为Linux软件生态做贡献的开发者们！

软件推荐

Linux日常软件推荐：

以上软件仅供参考，标注 （星火） 的软件推荐使用星火商店下载，标注 wine 的软件推荐星火商店下载，其它的没有标注的软件随便

注：星火商店只支持Ubuntu20.04以上的系统，所以要体验星火商店的用Ubuntu20.04以上的即可

安装命令:

有些软件因为下载的比较慢，所以推荐在星火商店里下载

有些软件在终端里无法搜索出来，这时候可以尝试在新立得软件包管理器里面搜索，新立得软件包管理器也搜索不了的，就推荐在官网下载或者星火商店里面下载，如果你没有网络限制，那就推荐在官网下载，软件商店里的有些软件有点老旧，看自己取舍。

IntelliJ Idea、Pytharm这两个软件在官网中下载了之后解压，软件包里面有个bin文件夹，在该文件夹里面执行启动命令

IntelliJ Idea的启动方法：IntelliJ Idea下载之后解压里面有个bin文件夹，在bin文件夹里面打开终端，输入 ./idea.sh ，在回车即可启动IntelliJ Idea软件

Pytharm的启动方法：Pytharm下载之后解压里面有个bin文件夹，在bin文件夹里面打开终端，输入 ./pytharm.sh ，在回车即可启动Pytharm软件

vim安装

sudo apt-get install vim

sublime text安装

官方有详细的文档

VirtualBox安装

sudo apt-get install virtualbox

VMware虚拟机安装

首先在官方下载VMware虚拟机的执行文件，其后缀名为 .bundle ，然后在你下载bundle的文件夹下打开终端，输入 sudo ./你bundle文件的全称 ，例如：我下载的VMware虚拟机的名字叫 VMware-Player-Full-16.2.4-20089737.x86_64.bundle 那么就在终端输入 sudo ./VMware-Player-Full-16.2.4-20089737.x86_64.bundle 就可以执行安装程序了，这样，VMware虚拟机就自动安装了

WPS安装

下载了WPS的软件包之后，在软件包的位置打开终端，在终端执行 sudo apt-get install ./WPS的全称

VLC安装

sudo apt-get install vlc

gThumb安装

sudo apt-get install gThumb

Google浏览器安装

在官网下载Google浏览器软件包之后，在软件包的位置打开终端，在终端输入 sudo apt-get install ./Google浏览器的全称

其他的软件以此类推，下载的软件包，在软件包位置打开终端，输入 sudo apt-get install ./软件包全称

在终端下载软件则输入 sudo apt-get install 软件包的名称，注意大小写，如果终端里搜索不到软件包，那么就在新立得软件包管理器里面搜索下载，新立得软件包管理器里面也没有的那就看看星火商店，星火商店里面也没有的就去官网
