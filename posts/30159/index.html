VMware设置静态IP

发表于2023-10-04|Linux

开始前准备

打开终端，在终端中输入 ipconfig 查看VMware的NAT网卡IP，记下来，等下要用到

取消勾选DHCP服务（取消DHCP的缺点就是每次创建新的虚拟机都需要自己手动添加IP地址，否则无法联网）

感觉因该可以不需要关闭DHCP服务，大不了每次开机看一下 IP

这个也是，要记下来，等下会用到

给服务器配置静态IP地址

到现在系统无法使用网络是因为还没有设置IP，所以我们要自己配置一下，首先进入到 cd /etc/sysconfig/network-scripts/ 目录下，然后输入命令 ll 或者 ls ，查看当前目录下的文件，我这里显示的是ifcfg-ens33，这个是网卡配置文件，有些网卡配置文件的名称有可能是ifcfg-ens32，也有可能是ifcfg-ens1667777，总之 ifcfg-ens 开头的这个文件，差不多就是自己服务器网卡的配置文件

用 vi 命令来编辑自己的网卡配置文件，有些网卡的名字可能不一样（可能是 ifcfg-enp0s3 之类的，但都是 ifcfg 开头的），我这里的网卡名称是 ifcfg-ens33 ， 所以输入 vi ifcfg-ens33

输入内容：

静态IP地址推荐使用网关IP，但是不是直接使用，而是最后三位数要修改一下，打个比方，我的网关IP地址为 192.168.205.2 ，那么我的静态IP就可以是 192.168.205.103 ，也就是把 2 那里修改一下即可

如果你有多个虚拟机，那么在修改静态IP的时候就不要和别的虚拟机IP重叠了，也就是说我的另一个虚拟机IP是 192.168.205.103 ，那么我在修改静态IP的时候就不能使用 192.168.205.103 了，也就是除了这个 192.168.205.103 之外其它的IP地址都可以，比如 192.168.205.104

修改好之后按 ESC 键，输入 :wq! 回车即可保存文件并退出，然后在输入命令行 systemctl restart network 重启一下网络，在输入 ip addr 即可查看自己的 IP 地址

默认网关那里有就写，没有就不写，配置完之后一路点击下面的确定即可

最后在 ping 一下，如果 ping 的通的话就说明网络连接正常

然后就可以使用SSH工具连接自己的VMware本地服务器了，命令行：ssh 用户名@IP地址

我要使用 ROOT 账户，我的IP又是 192.168.205.207，所以我这里要输入 ssh root@192.168.205.207
