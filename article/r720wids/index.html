VMware设置静态IP

555字约2分钟

Linux命令

2023-10-4

打开终端，在终端中输入 ipconfig 查看VMware的NAT网卡IP，记下来，等下要用到

取消勾选DHCP服务（取消DHCP的缺点就是每次创建新的虚拟机都需要自己手动添加IP地址，否则无法联网）

感觉因该可以不需要关闭DHCP服务，大不了每次开机看一下 IP

这个也是，要记下来，等下会用到

配置静态IP地址

❗❗❗因为已经关闭DHCP服务，所以每个系统都需要自己手动分配IP，注意不要IP冲突❗❗❗

我这里详细讲一下CentOS7设置IP的方法

进入到 cd /etc/sysconfig/network-scripts/ 目录下，查看网卡配置文件，我这里显示的是ifcfg-ens33，这个是网卡配置文件

有些网卡配置文件的名称有可能是ifcfg-ens32，也有可能是ifcfg-ens1667777，总之 ifcfg-ens 开头的这个文件，差不多就是自己服务器网卡的配置文件

编辑自己的网卡配置文件

输入内容：

IPADDR=192.168.205.207

NETMASK=255.255.255.0

GATEWAY=192.168.205.2

DNS=192.168.205.2

注意自己的网关IP，我的网关IP为 192.168.205.2

那么我的静态IP就为是 192.168.205.103，第四节数字可以随便改，范围在1~255 ，注意不要IP冲突

修改好之后按 ESC 键，输入 :wq! 回车即可保存文件并退出

输入命令行 systemctl restart network 重启一下网络，在输入 ip addr 即可查看自己的 IP 地址

默认网关那里有就写，没有就不写，配置完之后一路点击下面的确定即可

最后在 ping 一下，如果 ping 的通的话就说明网络连接正常
