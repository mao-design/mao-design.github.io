Ubuntu安装星火软件软件

417字约1分钟

星火商店

2022-4-15

安装星火软件商店

星火软件商店官网：星火应用商店

第一步，安装依赖，如果你是Deepin、UOS或者Ubuntu22.04系统，则不需要安装依赖文件

点击星火软件商店下载按钮，则会自动跳转到 gitee 上

注意：如果你是Deepin、UOS或者Ubuntu22.04系统，则不需要安装依赖文件，直接安装即可

官网下载的依赖包里面有两个文件夹，一个是Debian10和Ubuntu20.04的依赖包，一个是Debian11的依赖包，这两个依赖包文件夹里面都有一个 一键安装.sh 的脚本，在依赖包文件目录里面打开终端，输入 sudo ./一键安装.sh 即可运行 sh 脚本，这时依赖则会自动安装。

依赖安装完成之后再安装星火软件商店，先进入到星火软件商店软件包的下载目录里面，打开终端，输入 sudo apt install ./星火软件商店的全名 ，比如星火软件商店的软件包全名叫 spark-store_3.3.0.3_amd64.deb ，那么终端中输入的命令则是 sudo apt install ./spark-store_3.3.0.3_amd64.deb ,然后等待安装即可，安装完成之后就可以在星火软件商店里面安装软件了，也再此感谢那些为Linux软件生态做贡献的开发者们！
