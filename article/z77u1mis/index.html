npm图床

422字约1分钟

Hexonpm

2024-3-12

参考：npm图床的使用技巧 | Akilarの糖果屋

安装node.js 环境

注册npm的账号，访问npm注册页面

添加本地npm用户设置

# 仅第一次使用需要添加用户，之后会提示你输入你的npm账号密码以及注册邮箱
npm adduser
# 非第一次使用直接登录即可，之后会提示你输入你的npm账号密码以及注册邮箱
npm login

到npm官网搜索自己想定义的包名，如果搜不到则表示该包名没有被占用，那么你就可以使用该包名来定义你的npm包的名字

初始化npm

npm init

最后提示Is this OK? (yes)输入yes

将自己的包发布到npm上

注意：npm包的版本号不能重复，所以每次更新npm包的时候都需要更新一下版本号

# 更新版本号，在现在版本的基础上 + 1
npm version patch
# 发布npm包
npm publish

访问资源（站点推荐）

# 我使用的是第一个
https://jsd.onmicrosoft.cn/npm/package@version/file
https://gcore.jsdelivr.net/npm/package@version/file
https://unpkg.com/package@version/file

package 你的npm包名

version 你所访问的文件版本号（也就是你上传该文件的时候是什么版本就写什么版本，版本号不能写错，当然，也可以不用版本号，官方的解释是带版本号的要比没有带版本号的快一点，看自己吧）

例子：

我的npm包：mao-design-server

我要访问我npm包下的 /img/森林.png图片

https://jsd.onmicrosoft.cn/npm/mao-design-server/img/森林.png
